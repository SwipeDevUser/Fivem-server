# Admin Resource

Admin commands and management tools.

## Features

- Admin commands
- Player management
- Server control
- Logging and monitoring

## Setup

1. Ensure dependencies are loaded: `core`
2. Add `ensure admin` to your `server.cfg`
3. Configure admin permissions in ACE list

## Commands

- `/kick [id]` - Kick player
- `/ban [id]` - Ban player
- `/mute [id]` - Mute player
- `/jail [id]` - Jail player
- `/admin` - Open admin menu

## Permissions

Requires `admin.kick`, `admin.ban`, `admin.mute` ACE permissions
