# Identity Resource

Player identity and character management system for FiveM.

## Features

- Player character creation
- Identity retrieval
- Character persistence
- Account management

## Setup

1. Ensure dependencies are loaded: `core`
2. Add `ensure identity` to your `server.cfg`

## Dependencies

- core
- ox_lib

## API

### Exports

- `getPlayerIdentity(source)` - Get player's identity data
