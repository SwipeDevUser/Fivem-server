# UI Resource

User interface and NUI system.

## Features

- NUI interface
- UI components
- HUD elements
- Notifications

## Setup

1. Ensure dependencies are loaded: `core`
2. Add `ensure ui` to your `server.cfg`

## Files

- `html/index.html` - Main UI page
- `html/style.css` - Styling
- `html/script.js` - Client-side scripts

## Commands

- `/ui` - Show UI

## API

### Exports

- `updateUI(source, data)` - Update UI for player
